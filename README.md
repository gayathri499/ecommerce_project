Full Stack E-Commerce Web Application with product search, cart system, and user authentication using HTML, CSS, JavaScript, Node.js, and Express.
# 🛒 Simple E-Commerce Web Application

## 📌 Project Description
This is a full stack e-commerce web application that allows users to browse products, search items, add them to a cart, and perform user authentication (signup/login).

The project is built using basic frontend technologies and a Node.js backend, making it beginner-friendly and easy to understand.

---

## 🚀 Features

### 🛍️ Product Features
- Display product list with images
- Search products dynamically
- Responsive product grid layout

### 🛒 Cart Features
- Add to cart
- Remove items from cart
- Increase / decrease quantity
- Auto-save cart using localStorage

### 🔐 Authentication
- User Signup
- User Login
- Basic authentication using backend

### Backend
